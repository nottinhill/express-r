# express-r

Express Router to call R studio scripts. 

## Install

### Optional

`export PORT=1234`

### Mandatory
```
npm install
npm start
```


## Usage 

`curl -X POST http://localhost.save-up.net:8444/r -d '{"foo":"bar","john":"doe"}' --header 'Content-Type: application/json'`
